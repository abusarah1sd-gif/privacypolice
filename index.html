Kebijakan Privasi – Chester Friends

Kebijakan Privasi ini menjelaskan bagaimana game seluler Chester Friends (“kami”, “game”, atau “aplikasi”) menangani informasi pengguna. Chester Friends adalah game offline yang tidak mengumpulkan, menyimpan, atau membagikan data pribadi apa pun.

1. Informasi yang Kami Kumpulkan

Chester Friends tidak mengumpulkan informasi pribadi dalam bentuk apa pun.

  • Tidak memerlukan akun atau login
  • Tidak mengumpulkan nama, email, atau identitas pengguna lainnya
  • Tidak mengumpulkan data lokasi
  • Tidak mengirim atau menyimpan data ke server mana pun

Beberapa informasi non-pribadi seperti tipe perangkat atau versi sistem operasi mungkin dihasilkan oleh perangkat Anda tetapi tidak dikumpulkan atau disimpan oleh kami.

2. Tidak Menggunakan Layanan Pihak Ketiga

Game ini tidak menggunakan:

  • AdMob atau layanan periklanan apa pun
  • Google Analytics, Firebase Analytics, atau alat analitik lainnya
  • SDK pelacakan pihak ketiga
  • Penyimpanan cloud atau server eksternal

Chester Friends berjalan sepenuhnya offline.

3. Privasi Anak

Chester Friends aman digunakan oleh semua usia. Karena game ini tidak mengumpulkan data pribadi, game ini mematuhi aturan privasi anak termasuk COPPA.

4. Keamanan Data

Karena kami tidak mengumpulkan atau menyimpan data pengguna, tidak ada risiko data disalahgunakan atau hilang.

5. Perubahan pada Kebijakan Privasi Ini

Kami dapat memperbarui Kebijakan Privasi ini jika diperlukan. Perubahan akan dipublikasikan pada halaman ini dengan tanggal yang diperbarui.
